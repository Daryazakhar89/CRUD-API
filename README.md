# CRUD-API

**Installation**

```sh
https://github.com/Daryazakhar89/CRUD-API.git
```

```sh
npm install
```


**Configuration**

Scripts

```sh
npm run start:dev
```
```sh
npm run start:prod
```
```sh
npm run start:multi
```
```sh
npm run test
```

Script *npm run test* runs tests for all methods with valid and invalid data. For manual testing Postman, Insomnia or other similar tools can be used.
